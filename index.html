slm.js

slm.js makes easy to create complex dynamic layout for your web pages.
Simply add jQuery and slm.js in the header and call the plugin on the root element in the document.ready callback.
You have several ( 5 by now) layout managers available that will handle the geometry and the resize automatically.
You simply have to configure the layout through the "layout" attribute keeping your HTML code clean and readable without complex combinations of css classes.

Cool! But the amazing part is the source code that you need in order to obtain this.

<div id="root" layout="sz:'splitV',sash:100" style="width:100%;height:300px">
    <div style="background:cyan">Cyan</div>
    <div layout="ex:1,sz:'splitH',sash:100">
        <div style="background:magenta">Magenta</div>
        <div layout="ex:1,sz:'splitV',sash:100">
            <div style="background:yellow">Yellow</div>
            <div layout="ex:1,sz:'splitH',sash:100">
                <div style="background:orange">Orange</div>
                <div style="background:lightgreen">Green</div>
            </div>
        </div>
    </div>
</div>
