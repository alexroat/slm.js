slm.js

slm.js makes easy to create complex dynamic layout for your web pages.
Simply add jQuery and slm.js in the header and call the plugin on the root element in the document.ready callback.
You have several ( 5 by now) layout manager available that will handle the geometry and the resize automatically.
You simply have to configure the layout through the "layout" attribute keeping your HTML code clean and readable without complex combinations of css classes.

Try the demo here and check the source code to see how much simple and obvious is it.
